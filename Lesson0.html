Guide to Vim: Lesson 0

The Absolute Basics

Vim is different than other editors you have used. It is modal. By default, keys you hit do not go into the document you are editing! Instead, they trigger commands. So the first thing you need to do is figure out how to enter text, stop entering text, and save your file.

The Four Most Important Commands

  • i: shift into insert mode, and actually be able to enter text!
  • Esc: the escape key exits insert mode, and puts you back in command mode.
  • x: delete character under the cursor.
  • :wq: in command mode, this writes your current file to disk and exits vim.

Homework

Call up a practice file in vim (vim practice.txt) and practice each of the above four commands until your fingers find them almost automatically.
